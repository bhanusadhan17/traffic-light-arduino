# traffic-light-arduino
Traffic light controller using Arduino Uno with LED sequencing
## Components Used
- Arduino Uno
- 3x LEDs (Red, Yellow, Green)
- 3x 220 ohm resistors
- 3x 220 ohm resistors
- Breadboard
- Jumper wires

## Pin Connections
- Red LED → Pin 8
- Yellow LED → Pin 9
- Green LED → Pin 10

## How It Works
LEDs cycle through Red → Yellow → Green with delays.
Simulates a real traffic light sequence.

## Tools Used
- Arduino IDE
- Tinkercad (simulation)
